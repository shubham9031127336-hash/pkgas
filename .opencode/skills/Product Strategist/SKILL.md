# Nutan Gases Project Context

This skill is configured for **Nutan Gases** — a gas cylinder refilling and inventory management system in Khagaria, Bihar, India.

**The product has three facets:**
1. **Public marketing site** — SEO-driven landing pages, blog, lead capture, cylinder serial tracker
2. **Admin panel** — Full ERP with inventory management, order creation wizard, customer CRUD, cylinder lifecycle tracking, partner exchange, payments, AI assistant, multi-business support (Nutan Gases + VD Enterprises)
3. **Customer portal** — Self-service dashboard, order history, cylinder tracking, payments, profile management

**Business model:** Industrial and medical gas supply (B2B). Two business entities (Nutan Gases, VD Enterprises) under one account. Revenue from gas refilling, cylinder deposits, and rental cylinders.

**Key differentiators:** AI-powered assistant for staff, real-time cylinder serial tracking, multi-business support in one system, customer self-service portal.

**Competitive context:** Local gas suppliers in Upper Bihar (Khagaria, Dibrugarh, Duliajan, Digboi) — mostly paper-based or using generic billing software. Nutan Gases' digital-first approach is a market differentiator.

# Pipeline Integration

## Pipeline Role
This skill is **Step 2 of 6** in the design pipeline: `UX Researcher → Product Strategist → Wireframe Generator → UI Designer → Design Critic → UX Audit Skill`

## Pipeline Input
Expects structured output from **UX Researcher**: product_context, user_personas, pain_points, behavioral_insights, opportunity_areas, research_summary. When no research is provided, default to the Nutan Gases project context.

## Pipeline Output
After defining strategy, output structured data as JSON with:
- `product_vision`: mission, value proposition, differentiation (must reference gas cylinder management domain)
- `target_audience`: refined audience segments (admin roles, portal customers, public visitors)
- `competitive_landscape`: competitors analyzed (local gas suppliers, generic billing software), positioning, market gaps
- `feature_priorities`: prioritized features with rationale and framework scores (order wizard, cylinder tracking, AI assistant, portal, etc.)
- `roadmap`: MVP, V1, growth phases with milestones (grounded in Nutan Gases feature set)
- `success_metrics`: north star metric, KPIs (cylinder utilization, refill cycle time, deposit balance, inventory turnover)
- `strategy_summary`: concise strategy overview

## Pipeline Handoff
Pass the output JSON to **Wireframe Generator** with: "Strategy defined for Nutan Gases — generate wireframes and IA based on this product strategy."
